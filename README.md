# my-first-NestJS-app
A demo NestJS app about a Pizzeria

##

Api Routes:

- `POST` - `/auth`
- `DELETE` - `/auth`

- `GET` - `/admin/profile`
- `GET` - `/admin/profile/:id`
- `POST` - `/admin/profile`
- `PUT` - `/admin/profile/:id`

- `POST` - `/admin/pizza-components`
- `PUT` - `/admin/pizza-components/:id`

- `GET` - `/admin/order`
- `POST` - `/admin/order`
- `PUT` - `/admin/order/:id`

- `GET` - `/profile`
- `PUT` - `/profile`

- `GET` - `/pizza-components`
- `GET` - `/pizza-components/:type`

- `GET` - `/order`
- `POST` - `/order`
- `PUT` - `/order/:id`
